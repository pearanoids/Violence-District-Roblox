## ⚙️ Cara Penggunaan

### 🔧 Langkah 1 — Persiapan
Pastikan kamu sudah memiliki executor Roblox yang terinstall di perangkatmu.
> Executor yang direkomendasikan: **Delta**, **Arceus X**, **Fluxus**, atau **Hydrogen**

---

### 🎮 Langkah 2 — Buka Game
Masuk ke Roblox dan buka game **Violence District**.

---

### 📋 Langkah 3 — Copy Script

**Metode Loadstring:**
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/pearanoids/Violence-District-Roblox/refs/heads/main/SBX_Script.lua"))()
```
